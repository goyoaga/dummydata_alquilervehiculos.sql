# Dummy Data SQL - Alquiler de Vehículos

Este repositorio contiene datos de ejemplo para un sistema de alquiler de vehículos.  
Se incluyen tablas de vehículos, clientes y alquileres.

## Cómo usar
1. Crear una base de datos nueva en MySQL o PostgreSQL.
2. Ejecutar el script `dummydata-sql` para crear las tablas y rellenarlas con datos de prueba.

## Tablas incluidas
- **vehiculos**: Información de los vehículos (marca, modelo, año, capacidad de pasajeros y carga).
- **clientes**: Información de los clientes (nombre, apellido, DNI, dirección, teléfono).
- **alquileres**: Relación de alquileres entre vehículos y clientes, con fechas y precio total.
